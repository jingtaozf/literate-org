* What this is

A Python runtime that lets you write code in =literate-org.org= and
=import= it as if it were a normal Python package — both via on-the-fly
loading from the org file *and* via tangled =literate_python/*.py= for
consumers that don't read org. An optional Flask server (=make server=)
exposes live exec + cross-module hot reload for Emacs and other
editor-side clients.

The project also ships its own LP discipline:
- =.claude/rules/*.md= rules for AI agents working in this repo
- =.claude/hooks/= PreToolUse + PostToolUse hooks for auto-tangle
  and protection of generated artefacts
- =scripts/check_org_structure.py= / =check_nl_outline.py= as fitness
  functions for the literate document
- Six pre-commit gates: black, flake8, check-structure,
  check-nl-outline, check-dead-code, gitleaks

* Quick start

Install:
#+BEGIN_SRC bash
poetry install
#+END_SRC

Run the literate-server:
#+BEGIN_SRC bash
make server   # Flask on $LITERATE_PYTHON_HOST:$LITERATE_PYTHON_PORT (default 127.0.0.1:7330)
#+END_SRC

Tangle the org file (auto-run by the PostToolUse hook on any =.org= edit;
this is the manual fallback):
#+BEGIN_SRC bash
make tangle FILE=literate-org.org
#+END_SRC

Run lints / tests:
#+BEGIN_SRC bash
make check-structure   # depth ≤ 5, no grab-bag, prose-first
make check-nl-outline  # ≥40-line functions need # # outline comments
make check-dead-code   # vulture
make secret-scan       # gitleaks across all files
make lint              # black + flake8
poetry run pytest literate_python/tests/
#+END_SRC

* Minimum example

The project itself is the worked example — =literate-org.org= contains
the full module set as 62 code blocks across ~3300 lines of prose.
Module sections are identified by a =:LITERATE_ORG_MODULE:= property
and tangle to =./literate_python/<module-path>.py=. From Python:

#+BEGIN_SRC python
from literate_python import server  # ← imported via the standard
                                    #   sys.meta_path mechanism;
                                    #   resolution goes through
                                    #   LiterateImporter.
#+END_SRC

Once a module is registered (via =register_literate_modules()= or by
loading =literate-org.org= directly with =literate-elisp-load=), it is
indistinguishable from a hand-written package to the rest of Python.
